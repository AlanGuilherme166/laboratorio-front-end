Formatação no HTML serve para exibir outas formas de texto, neste caso, o Negrito 

            O elemento HTML <b> representa um intervalo de texto estilísticamente diferente do texto normal.
            Ele é geralmente usado para destacar palavras-chaves em um resumo,
            nomes de produtos em um comentário ou outros vãos de texto cuja a apresentação típica seria negrito.
